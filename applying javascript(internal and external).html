applying javascript(internal and external)

Internal JavaScript:

JavaScript can be added directly to the HTML file by writing the code inside the <script> tag . We can place the <script> tag either inside <head> or the <body> tag according to the need.

<html>
<head>
<title>Internal JavaScript</title>
<script type="text/javascript">
document.write("This is Internal Javascript Example.!!!");
</script>
</head>
<body>
</body>
</html>

Output:

This is Internal Javascript Example.!!!

External Javascript:

The other way is to write JavaScript code in another file having a .js extension and then link the file inside the <head> or <body> tag of the HTML file in which we want to add this code.

HTML Code:

<!DOCTYPE html>
<html>
<head>
<title>External JavaScript</title>
<style>
h2 {
box-sizing: border-box;
width: 250px;
height: 150px;
background-color: green;
color: white;
margin: auto;
font-size: 15px;
font-family: sans-serif;
text-align: center;
COMPUTER SCIENCE & ENGINEERING SKILL ORIENTED LAB - II
padding: 20px;
}
</style>
</head>
<body>
<h2 class="external">
BVC College of engineering</h2>
<script src="BVC.js">
</script>
</body>
</html>
JS Code:
let h2 = document.querySelector(".external");
h2.addEventListener("mouseenter", function (e) {
h2.innerText = "Hi STUDENTS, Welcome to BVC";});

Output:

Bvc College of Enginnering
Hi STUDENTS, Welcome to BVC
